rock-paper-scissors

This is my attempt at the "Rock Paper Scissors" exercise for The Odin Project, under the Foundations Course, in the Javascript section.

The file contains a README.md, index.html, attributes.html, separate stylesheets for phones and laptops, game.js and the folder of images used.

Technologies used to implement: HTML5, CSS3, Javascript.

Expecting to- Create a fun, lightweight website to play rock, paper, scissors on.

Concepts learnt- Using (and significance of) defer, using Javascript loops, Conditionals and functions, Taking user input in Javascript through buttons and prompts, Event listeners, Adding and removing classes to elements using Javascript, Image manipulation with CSS, CSS positions, Media queries, Adaptive Design.
